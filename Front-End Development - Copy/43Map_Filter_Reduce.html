Map, Filter, and Reduce in Java Script.

1. map() Method

It creates a new array by applying a function to each element of the original array.

(It doesn't change the original array).

Explanation.

The map() goes through each element of numbers.

Applies num * 2 to each.

Returns a new array [2, 4, 6, 8, 10].

2. filter() Method

It creates a new array contaning only the elements that pass a certain test(condition).

Explanation.

filter() checks each num.

Only keeps values where the condition num > 30 is true.

Returns a new array [45, 60].

3. reduce() Method

It reduces the array to a single value (like... sum, average, or object).

It runs a function on each element while keeping an accumulator.

Explanation.

acc starts from 0 (initial value).

On each step, adds the current num to acc.
