Spread operator in Java Script.

ChatGPT said: The Spread Operator (...) in JavaScript is used to expand (spread) the elements of an iterable (like an array, string, or object) into individual elements.

It makes it easier to copy, combine, or pass elements without manually looping.

Syntax:

...iterable

Examples in short.

    Use Case	        Example	                Purpose  

    Copy array	        const b = [...a]        Create shallow copy 

    Merge arrays	[...a, ...b]	        Combine elements 

    Copy object	        {...obj}	        Clone object 

    Merge objects	{...obj1, ...obj2}	Combine key-values 

    Function arguments	fn(...arr)	        Pass array as args
