1. What is a Higher Order Function?

> A Higher Order Function (HOF) is a function that does one or both of the following:

1.Takes another function as an argument (callback)

2.Returns a function

This makes functions in JavaScript first-class citizens (they can be passed around just like values).

2.What is callback?

A callback is simply a function passed as an argument to another function, which is later called back inside that function.
