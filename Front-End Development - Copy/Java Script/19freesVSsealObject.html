Object.freeze( ) vs Object.seal( )

Seal

Object.seal() prevents adding or removing properties from an object, but allows modifying existing properties. 

const sealedObject = {
  name: "Ankit",
  age: 20
};

Object.seal(sealedObject);

    // This will work (modifying existing property)
    sealedObject.age = 21;

    // These will not work (in non-strict mode they fail silently)
    sealedObject.city = "japan"; // Cannot add new property
    delete sealedObject.name;       // Cannot delete property

    // Check if object is sealed
    console.log(Object.isSealed(sealedObject)); // true

freeze

Object.freeze() is more restrictive - it prevents adding, removing, or modifying properties.

const frozenObject = {
  name: "piyush",
  age: 20
};

Object.freeze(frozenObject);

    // These will not work (in non-strict mode they fail silently)
    frozenObject.age = 21;          // Cannot modify existing property
    frozenObject.city = "japan"; // Cannot add new property
    delete frozenObject.name;       // Cannot delete property

    // Check if object is frozen
    console.log(Object.isFrozen(frozenObject)); // true
