Deep copy & Shallow Copy in JS.

Shallow Copy

A shallow copy creates a new object and copies all the top-level properties of the original object. However, if a property is itself an object (or an array, which is a type of object), it copies the reference to that nested object, not the object itself.

The result: The top-level properties are independent, but the nested objects are shared between the original and the copy. Modifying a nested object in one will affect the other.

Deep copy

A deep copy creates a completely new object and then recursively copies all properties of the original object, including all nested objects. The original and the copy are entirely independent of each other at all levels. No changes to one will affect the other.
