String method & properties

Properties

1. length
Returns the number of characters in a string.

const str = "Hello";
console.log(str.length); // 5

Method

2. charAt() / []
Returns the character at a specified index.

javascript
const str = "Hello";
console.log(str.charAt(1)); // "e"
console.log(str[1]);        // "e" (modern approach)

3. charCodeAt() / codePointAt()
Returns the Unicode value of a character.

javascript
const str = "Hello";
console.log(str.charCodeAt(0)); // 72 (H)
console.log(str.codePointAt(0)); // 72

4. concat()
Combines two or more strings.

javascript
const str1 = "Hello";
const str2 = "World";
console.log(str1.concat(" ", str2)); // "Hello World"

5. includes()
Checks if a string contains a specified substring.

javascript
const str = "Hello World";
console.log(str.includes("World")); // true

6. indexOf() / lastIndexOf()
Returns the position of the first/last occurrence of a substring.

javascript
const str = "Hello World";
console.log(str.indexOf("o"));      // 4
console.log(str.lastIndexOf("o"));  // 7

7. startsWith() / endsWith()
Checks if a string starts/ends with a specified substring.

javascript
const str = "Hello World";
console.log(str.startsWith("Hello")); // true
console.log(str.endsWith("World"));   // true

8. slice() / substring() / substr()
Extracts parts of a string.

javascript
const str = "Hello World";
console.log(str.slice(0, 5));      // "Hello"
console.log(str.substring(0, 5));  // "Hello"
console.log(str.substr(6, 5));     // "World"

9. split()
Splits a string into an array of substrings.

javascript
const str = "apple,banana,cherry";
console.log(str.split(",")); // ["apple", "banana", "cherry"]

10. toLowerCase() / toUpperCase()
Converts case.

javascript
const str = "Hello";
console.log(str.toLowerCase()); // "hello"
console.log(str.toUpperCase()); // "HELLO"

11. trim() / trimStart() / trimEnd()
Removes whitespace.

javascript
const str = "  Hello  ";
console.log(str.trim());      // "Hello"
console.log(str.trimStart()); // "Hello  "
console.log(str.trimEnd());   // "  Hello"

12. replace() / replaceAll()
Replaces substrings.

javascript
const str = "Hello World";
console.log(str.replace("World", "JavaScript")); // "Hello JavaScript"
console.log(str.replaceAll("l", "L"));           // "HeLLo WorLd"

13. padStart() / padEnd()
Pads a string to a certain length.

javascript
const str = "5";
console.log(str.padStart(3, "0")); // "005"
console.log(str.padEnd(3, "0"));   // "500"

14. repeat()
Repeats a string.

javascript
const str = "Hello";
console.log(str.repeat(3)); // "HelloHelloHello"

15. search()
Searches for a match using a regular expression.

javascript
const str = "Hello World";
console.log(str.search(/World/)); // 6

16. match() / matchAll()
Returns matches against a regular expression.

javascript
const str = "Hello World";
console.log(str.match(/[A-Z]/g)); // ["H", "W"]

17. localeCompare()
Compares two strings in the current locale.

javascript
const str1 = "apple";
const str2 = "banana";
console.log(str1.localeCompare(str2)); // -1 (apple comes before banana)
Template Literal Methods

18. String.raw()
Returns a raw string from template literal.

javascript
const path = String.raw`C:\Development\profile\aboutme.html`;
console.log(path); // "C:\Development\profile\aboutme.html"
Modern Methods (ES6+)

19. normalize()
Returns the Unicode Normalization Form of a string.

javascript
const str = "\u1E9B\u0323";
console.log(str.normalize("NFC")); // "ẛ̣"

20. fromCharCode() / fromCodePoint() (Static methods)
Creates a string from Unicode values.

javascript
console.log(String.fromCharCode(72, 101, 108, 108, 111)); // "Hello"
console.log(String.fromCodePoint(9731, 9733, 9842));      // "☃★♲"
Example Usage
javascript
const text = "   JavaScript is awesome!   ";

// Chaining methods
const processed = text
  .trim()
  .toUpperCase()
  .replace("AWESOME", "AMAZING")
  .padEnd(30, "!");

console.log(processed); // "JAVASCRIPT IS AMAZING!!!!!!"
