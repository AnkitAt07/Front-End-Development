default parameters allow to set a default value for function parameters.

If no value is passed for that parameter (or undefined is passed), the default value will be used.

Basic Syntax

   function greet(name = "Guest")
     {
        console.log("Hello " + name);
     }
        greet();            // Output: Hello Guest
        greet("Piyush");    // Output: Hello Piyush

Explation of this code.

    name = "Guest" â†’ sets "Guest" as the default value.
    
    When greet() is called without any argument, it uses the default "Guest".
    
    When greet("Piyush") is called, the default value is replaced with "Piyush".
