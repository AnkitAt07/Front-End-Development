for of vs for in Loop in Java Script.

for....in loop

Purpose : Iterates over the keys/properties of an object (or indices of an array).

Works on : Objects, arrays (keys), strings (indices).

Returns : The property name (key/index), not the value.

for....of loop.

Purpose : Iterates over values of iterable objects like arrays, strings, sets, maps, etc.

Works on : Arrays, String, sets, maps (anything iterable).

Returns : The actual value, not the key.
