Truthy And Fasly in Js.

In JavaScript, a truthy value is a value that is considered true when encountered in a Boolean context. A falsy value is a value that is considered false in a Boolean context.

Falsy Value in JS.

There are only 8 falsy values in JavaScript. You should memorize them. Everything else is truthy.


false (The boolean false itself)

0 (The number zero)

-0 (Negative zero)

0n (BigInt zero)

"" (Empty string - single or double quotes)

null (The absence of any value)

undefined (A variable that has not been assigned a value)

NaN (Not a Number)

Truthy Value in JS.

Common Truthy Values

Since everything not on the falsy list is truthy, the list is endless. Here are the most common and sometimes surprising ones:

true (The boolean true)

"hello" (Any non-empty string, including "false" and "0")

42 (Any non-zero number, including negative ones, e.g., -1)

Infinity and -Infinity

[] (An empty array - this is a big one!)

{} (An empty object - this is another big one!)

function() {} (An empty function)

Dates, Regexes, etc.
