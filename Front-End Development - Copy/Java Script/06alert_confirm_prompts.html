Dialog Box.

A dialog box is a small window that pops up on a web page to display a message, ask for user input, or request confirmation. It forces the user to interact with it before they can continue using the rest of the page. This interaction is called "modal" behavior.

There are three types of dialog boxes.

1.alert

Purpose: To display a warning or informational message to the user.
Behavior: It has only an "OK" button. The script pauses until the user clicks OK.

2.confirm()

Purpose: To get a yes/no (true/false) answer from the user.

Behavior: It has an "OK" button and a "Cancel" button.

Clicking OK returns true.

Clicking Cancel returns false.

3.prompt()

Purpose: To get text input from the user.

Behavior: It has an input field, an "OK" button, and a "Cancel" button.

Clicking OK returns the text value from the input field (a string).

Clicking Cancel returns null
