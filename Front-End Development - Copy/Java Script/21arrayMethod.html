Most Common Array Methods in JavaScript

Summary Table.

    Method	Purpose	                    Mutates Original?	Returns
    
    map()	Transforms each element	                No	New Array
   
    filter()	Selects elements that pass a test	No	New Array
    
    find()	Finds first matching element	        No	Element
    
    some()	Checks if any element passes a test	No	Boolean
    
    every()	Checks if all elements pass a test	No	Boolean
    
    forEach()	Executes a function for each element	No	undefined
    
    reduce()	Reduces array to a single value	        No	Single Value
    
    slice()	Copies a portion of the array	        No	New Array
    
    concat()	Merges arrays                   	No	New Array
    
    push()	Adds element(s) to the end      	Yes	New Length
    
    pop()	Removes element from the end    	Yes	Removed Element
    
    splice()	Adds/removes elements at index  	Yes	Array of Removed Items
    
    sort()	Sorts the array	                        Yes	Sorted Original Array
    
    reverse()	Reverses the array	                Yes	Reversed Original Array
