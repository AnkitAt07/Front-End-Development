What is BOM in JavaScript?

BOM stands for Browser Object Model.

It allows JavaScript to interact with the browser, outside of the web page content (which is handled by the DOM – Document Object Model).



The Browser Object Model (BOM) provides everything JavaScript needs to interact with the browser environment itself – from managing URLs, alerts, and history, to getting browser and screen details. It complements the DOM by allowing control over the browser rather than just the web page.
