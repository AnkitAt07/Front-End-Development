Lexical and Block Scope in JS.

1. Lexical Scope in JavaScript

>   Definition: Lexical scope (also called static scope) means that the scope of a variable
    is determined by where it is written in the code, not where it is called from.

>   Inner functions can access variables defined in their outer functions because of lexical
    scope.

2. Block Scope in JavaScript

>   Definition: Block scope means that variables declared with 
    let or const are only accessible inside the block { } in which they are defined.

>   var does not follow block scope (it is function-scoped).
