Some and Every Array Method in JavaScript.

The some() and every() array methods are used to test elements in an array against a condition (a callback function). They return boolean values (true or false) and are non-mutating, meaning they donâ€™t change the original array.

1. Array.prototype.some()

Purpose:

Checks if at least one element in the array passes the test implemented by the callback function.

       Syntax:
            array.some(callback(element, index, array), thisArg)

Returns;

"true" if any element passes the test, otherwise "false".

2. Array.prototype.every()

Purpose:

Checks if all elements in the array pass the test implemented by the callback function.

       Syntax:
            array.every(callback(element, index, array), thisArg)

Returns:

true if all elements pass the test, otherwise false.
