Functions in Java script.

A function in JavaScript is a block of code designed to perform a specific task. Functions are executed when they are called (invoked).

Syntax

function functionName(parameters) {
        // code to be executed
        return result; // optional
}

Why We Use Functions?

        1.Reusability: Write once, use multiple times

        2.Modularity: Break complex problems into smaller pieces

        3.Readability: Make code more organized and understandable

        4.Maintainability: Easier to debug and update
